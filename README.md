# 🚀 Resume Spark

Resume Spark is an AI-powered resume builder designed to help users create professional, ATS-friendly resumes with an intuitive and responsive interface. It enables users to build, edit, and manage resumes efficiently while providing a seamless user experience.

## 🌐 Live Demo

🔗 https://resume-spark-380.lovable.app/

---

## ✨ Features

- 📄 Create and edit professional resumes
- 🎨 Clean and responsive user interface
- 👤 User authentication
- 💾 Secure cloud data storage
- ⚡ Fast and interactive dashboard
- 📱 Mobile-friendly design
- 🔒 Secure backend powered by Supabase

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/your-username/resume-spark.git
```

### Navigate to the project

```bash
cd resume-spark
```

### Install dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file in the project root and add:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### Start the development server

```bash
npm run dev
```

## 🗄️ Database

This project uses **Supabase** as the Backend-as-a-Service (BaaS), providing:

- User Authentication
- PostgreSQL Database
- Row Level Security (RLS)
- Stored Procedures (PL/pgSQL)
- Real-time capabilities
